# PVGround Website

Modern ve profesyonel güneş enerjisi tasarım platformu web sitesi.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Site `http://localhost:3000` adresinde açılacaktır.

## Build

```bash
npm run build
```

## Özellikler

- Modern ve responsive tasarım
- Framer Motion ile animasyonlar
- Tailwind CSS ile stil
- Görsel placeholder alanları (görselleri sonra ekleyebilirsiniz)
- SEO uyumlu yapı

## Görsel Ekleme

Görselleri `public/images/` klasörüne ekleyip, ilgili bileşenlerdeki placeholder alanlarını güncelleyin.
