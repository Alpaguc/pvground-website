# Bayrak Görselleri

Bu klasöre aşağıdaki bayrak görsellerini ekleyin:

## Gerekli Dosyalar

1. **turkey.svg** veya **turkey.png**
   - Türkiye bayrağı
   - Boyut: 24x24 veya 32x32 piksel önerilir
   - Format: SVG (tercih edilir) veya PNG

2. **uk.svg** veya **uk.png**
   - İngiltere/Birleşik Krallık bayrağı
   - Boyut: 24x24 veya 32x32 piksel önerilir
   - Format: SVG (tercih edilir) veya PNG

3. **germany.svg** veya **germany.png**
   - Almanya bayrağı
   - Boyut: 24x24 veya 32x32 piksel önerilir
   - Format: SVG (tercih edilir) veya PNG

## Önerilen Kaynaklar

- **Flag Icons**: https://flagicons.lipis.dev/
- **Country Flags**: https://www.countryflags.com/
- **SVG Flags**: https://github.com/lipis/flag-icons

## Dosya Adları

Dil seçicide kullanılan dosya adları:
- Türkçe: `/images/flags/turkey.svg`
- İngilizce: `/images/flags/uk.svg`
- Almanca: `/images/flags/germany.svg`

## Not

Eğer görsel yüklenemezse, dil kodu (TR, EN, DE) gösterilecektir.
